QuickLens Chrome Extension Steals Crypto

"QuickLens - Search Screen with Google Lens"라는 이름의 Chrome 확장 프로그램이 악성코드 유포 및 암호화폐 탈취 시도로 인해 웹 스토어에서 제거되었습니다. 수천 명의 사용자가 영향을 받았을 가능성이 있으며, 브라우저 확장 프로그램 보안에 대한 각별한 주의가 필요합니다.

Korean Tax Service Exposes Wallet Seed

대한민국 국세청(NTS)이 공식 보도자료를 통해 압수된 암호화폐 지갑의 복구 구문(mnemonic phrase)을 실수로 노출했습니다. 이로 인해 해커들이 약 64억 원(480만 달러) 상당의 암호화폐를 탈취하는 사고가 발생했습니다.

Previously Harmless Google API Keys Expose Gemini Data

Google Maps 등 클라이언트 측 코드에 포함되어 안전하다고 여겨졌던 기존 API 키들이 Gemini AI 어시스턴트 인증에 사용될 수 있음이 발견되었습니다. 이를 통해 비공개 데이터에 접근할 수 있는 취약점이 존재하므로 API 키 관리 전략의 수정이 필요합니다.
